radical

radical is a web application for exploring Chinese characters, focusing mainly on using graphical decompositions. The application is based on the Chinese characters decomposition project, which is hosted on Wikimedia Commons.

Code

radical is an open-source project and can be found on the creator's Github here.

This project was created using the following languages / technologies: HTML, CSS, JavaScript, Bootstrap, AngularJS, Python, and Flask.

About the creator

radical was written by John Fang, a college student majoring in computer science at the University of Texas at Austin.

He is also interested in learning Chinese and sometimes studies it in his free time.
